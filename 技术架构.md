# 技术架构

## 选型

| 层 | 技术 | 理由 |
|---|---|---|
| 前端 | Next.js 14 + TypeScript | SSR 对 SEO 友好，SaaS 标配 |
| 样式 | TailwindCSS | 快 |
| 后端 | Next.js API Routes | 前后端一体，减少复杂度 |
| 数据库 | SQLite (Prisma ORM) | MVP 不需要 PostgreSQL |
| AI | Claude API (内容生成) + OpenAI (关键词分析) | 双模型各取所长 |
| 支付 | Paddle | 支持中国个人，PayPal 备选 |
| 部署 | VPS (Hetzner) | $20/月，够用到 1000 用户 |
| 缓存 | Redis | 限流+会话 |

## 架构图

```
用户 → Cloudflare DNS → VPS (Nginx)
                         ├── Next.js (前端 + API)
                         ├── SQLite (Prisma)
                         ├── Redis (缓存/限流)
                         └── AI APIs (Claude / OpenAI)
```

## MVP 功能清单

| 优先级 | 功能 | 描述 |
|---|---|---|
| P0 | 博客文章生成 | 输关键词→输出完整 SEO 文章（标题+结构+正文+meta） |
| P0 | SEO 评分 | 对生成内容实时打分（关键词密度/可读性/标题质量） |
| P0 | 落地页 | 介绍+试用入口+定价 |
| P1 | 用户注册/登录 | NextAuth + 邮箱 |
| P1 | 付费墙 | Paddle 订阅 |
| P1 | 内容历史 | 用户历史记录 |
| P2 | 批量生成 | 一次生成多篇 |
| P2 | 多语言 | 非英语市场 |
